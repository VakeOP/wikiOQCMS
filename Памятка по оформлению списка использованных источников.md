Правила оформления списков литературы изменились. В соответствии с обновленным ГОСТ Р 7.0.100--2018 **теперь необходимо по-новому указывать разделительные знаки и добавлять в конце каждой записи специальную характеристику (например, «Текст : непосредственный»).** Чтобы упростить переход на актуальные требования, мы подготовили эту краткую памятку с основными правилами и примерами.

-   **Порядок:** Источники в списке располагаются либо в алфавитном порядке, либо по мере их упоминания в тексте.

-   **Пунктуация:** Между областями описания ставится знак «**точка и тире**» (. --). В конце каждой записи ставится **точка**.

-   **Автор:** Если у источника от 1 до 3 авторов, фамилия первого выносится в начало (заголовок записи) в формате Фамилия, И. О..

-   **Сведения об ответственности:** После заглавия через косую черту (/) указываются И. О. и фамилии всех авторов.

-   **Вид содержания:** В конце **каждой** записи указывается вид содержания и средство доступа.

| **Формулировка** | **Что это? (Вид содержания)** | **Как это использовать? (Средство доступа)** | **Когда применяется?** |
| -- Текст : непосредственный. | Текст (книга, статья) | Непосредственно (глазами, без устройств) | Печатные книги, статьи в журналах, отчеты. |
| -- Текст : электронный. | Текст (статья, сайт) | С помощью компьютера | Статьи на сайтах, электронные книги, PDF-документы. |
| -- Изображение : непосредственное. | Изображение (картина) | Непосредственно (глазами) | Картины, плакаты, печатные фотографии. |
| -- Изображение : электронное. | Изображение (фото) | С помощью компьютера | Цифровые фотографии, изображения на сайтах. |
| -- Устная речь : аудио. | Устная речь (голос) | С помощью аудиопроигрывателя | Аудиокниги, подкасты. |
| -- Музыка : аудио. | Музыка | С помощью аудиопроигрывателя | Музыкальные треки, альбомы. |

Примеры оформления типов источников:

-   **Книга печатная:**

Автор, И. О. Заглавие : сведения, относящиеся к заглавию / И. О. Автор. -- Место издания : Издательство, Год. -- Количество страниц. -- Текст : непосредственный.

Пример:

**Бешенцев, В. А. Подземные воды севера Западной Сибири (в пределах Ямало-Ненецкого нефтегазодобывающего региона) / В. А. Бешенцев, Т. В. Семенова. -- Тюмень : ТюмГНГУ, 2015. -- 224 с. -- Текст : непосредственный.**

-   **Статья в печатном журнале:**

Автор, И. О. Заглавие статьи / И. О. Автор // Название журнала. -- Год. -- Том, № номер. -- Страницы.

Пример:

**Галиулин, Р. В.** Риск загрязнения окружающей среды метанолом в газовой промышленности / Р. В. Галиулин, Р. А. Галиулина. -- Текст : непосредственный // Проблемы анализа риска. -- 2018. -- Т. 15, № 4. -- С. 54--59.

-   **Статья на сайте (электронная):**

Автор, И. О. Заглавие статьи / И. О. Автор. -- Текст : электронный // Название сайта. -- Год. -- URL: [ссылка] (дата обращения: ДД.ММ.ГГГГ).

Пример:

**Шарофутдинова, Н. П.** Исследование возможности использования нефтяных пластовых вод в сельском хозяйстве / Н. П. Шарофутдинова, [и др.]. -- Текст : электронный // Universum: технические науки : электронный научный журнал. -- 2022. -- № 2(95). -- URL: [https://7universum.com/ru/tech/archive/item/13068](https://www.google.com/url?sa=E&q=https://7universum.com/ru/tech/archive/item/13068) (дата обращения: 20.10.2023).

-   **Сайт целиком:**

Название сайта : [сайт]. -- Год начала работы. -- URL: [ссылка] (дата обращения: ДД.ММ.ГГГГ). -- Текст : электронный.

Пример:

**Хабр : [сайт]. -- 2006. --** URL: [https://habr.com/](https://www.google.com/url?sa=E&q=https://habr.com/) **(дата обращения: 20.10.2023). -- Текст : электронный.**

-   **Неопубликованный документ (отчет, регламент):**

Заглавие документа / Ответственная организация. -- Место создания, Год. -- Количество страниц. -- Текст : [непосредственный или электронный]. -- Неопубликованный документ.

Пример:

**Технологический регламент эксплуатации установки... / ООО «Газпром добыча Иркутск». -- Иркутск, 2025. -- 150 с. -- Текст : непосредственный. -- Неопубликованный документ.**

-   **Стандарт (ГОСТ):**

Обозначение стандарта. Заглавие : сведения о введении. -- Место издания : Издательство, Год. -- Количество страниц. -- Текст : непосредственный.

Пример:

**ГОСТ Р 7.0.100--2018. Система стандартов по информации, библиотечному и издательскому делу. Библиографическая запись. Библиографическое описание. Общие требования и правила составления : национальный стандарт Российской Федерации : издание официальное : дата введения 2019-07-01. -- Москва : Стандартинформ, 2018. -- 124 с. -- Текст : непосредственный.**
